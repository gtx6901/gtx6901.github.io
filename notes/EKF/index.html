# 扩展卡尔曼滤波 (Extended Kalman Filter)

# 0、卡尔曼滤波的局限

在上一篇笔记里，我们推导了标准卡尔曼滤波（KF）的完整流程。回顾它的两个核心方程：

xk=Fkxk−1+Gkuk+wkx_k = F_k x_{k-1} + G_k u_k + w_k xk​=Fk​xk−1​+Gk​uk​+wk​

zk=Hkxk+vkz_k = H_k x_k + v_k zk​=Hk​xk​+vk​

注意到：这两个方程都是线性的。状态转移是对 xk−1x_{k-1}xk−1​ 做矩阵乘法，观测也是对 xkx_kxk​ 做矩阵乘法。
然而，现实世界中的系统常常是非线性的。

例如我们用 (x,y,θ)(x,y,\theta)(x,y,θ) 三个量来描述机器人在平面中的位姿状态，在其运动方程和观测方程中便很容易出现三角函数，也就成为了非线性的。

这时，标准 KF 就直接失效了，因为它的整个推导都依赖 "线性 + 高斯" 的假设。

那怎么办？
把非线性函数在工作点附近做一阶泰勒展开，用局部线性化来近似它。
这就是扩展卡尔曼滤波（EKF）的核心思路。

# 1、系统建模（非线性版本）

EKF 处理的系统，状态转移和观测方程的一般形式为：

xk=f(xk−1,uk)+wk(状态转移方程)x_k = f(x_{k-1}, u_k) + w_k \quad \text{(状态转移方程)} xk​=f(xk−1​,uk​)+wk​(状态转移方程)

zk=h(xk)+vk(观测方程)z_k = h(x_k) + v_k \quad \text{(观测方程)} zk​=h(xk​)+vk​(观测方程)

其中：

  • f(⋅)f(\cdot)f(⋅) 是（可以非线性的）状态转移函数
  • h(⋅)h(\cdot)h(⋅) 是（可以非线性的）观测函数
  • 噪声假设不变：wk∼N(0,Qk)w_k \sim N(0, Q_k)wk​∼N(0,Qk​)，vk∼N(0,Rk)v_k \sim N(0, R_k)vk​∼N(0,Rk​)

和 KF 相比，唯一的变化是把矩阵乘法 Fkxk−1F_k x_{k-1}Fk​xk−1​、HkxkH_k x_kHk​xk​ 替换成了一般函数 f(⋅)f(\cdot)f(⋅)、h(⋅)h(\cdot)h(⋅)。

看起来只是换了个写法，但这一换，之前所有关于线性高斯的推导就全部失效了。
协方差如何传播？卡尔曼增益如何计算？这些都需要重新处理。

# 2、线性化：一阶泰勒展开

EKF 的核心手段是局部线性化。我们在当前估计值附近，对 fff 和 hhh 分别做一阶泰勒展开。

# 对状态转移函数 fff 展开

在先验估计 x^k−1∣k−1\hat{x}_{k-1|k-1}x^k−1∣k−1​ 处展开：

f(xk−1,uk)≈f(x^k−1∣k−1,uk)+∂f∂xk−1∣x^k−1∣k−1(xk−1−x^k−1∣k−1)f(x_{k-1}, u_k) \approx f(\hat{x}_{k-1|k-1}, u_k) + \frac{\partial f}{\partial x_{k-1}}\bigg|_{\hat{x}_{k-1|k-1}} (x_{k-1} - \hat{x}_{k-1|k-1}) f(xk−1​,uk​)≈f(x^k−1∣k−1​,uk​)+∂xk−1​∂f​∣∣∣∣∣​x^k−1∣k−1​​(xk−1​−x^k−1∣k−1​)

定义这个偏导数矩阵（雅可比矩阵）为：

Fk≜∂f∂xk−1∣x^k−1∣k−1F_k \triangleq \frac{\partial f}{\partial x_{k-1}}\bigg|_{\hat{x}_{k-1|k-1}} Fk​≜∂xk−1​∂f​∣∣∣∣∣​x^k−1∣k−1​​

这样，状态转移方程在局部就被近似为线性形式：

xk≈f(x^k−1∣k−1,uk)+Fk(xk−1−x^k−1∣k−1)+wkx_k \approx f(\hat{x}_{k-1|k-1}, u_k) + F_k (x_{k-1} - \hat{x}_{k-1|k-1}) + w_k xk​≈f(x^k−1∣k−1​,uk​)+Fk​(xk−1​−x^k−1∣k−1​)+wk​

# 对观测函数 hhh 展开

在先验估计 x^k∣k−1\hat{x}_{k|k-1}x^k∣k−1​ 处展开：

h(xk)≈h(x^k∣k−1)+∂h∂xk∣x^k∣k−1(xk−x^k∣k−1)h(x_k) \approx h(\hat{x}_{k|k-1}) + \frac{\partial h}{\partial x_k}\bigg|_{\hat{x}_{k|k-1}} (x_k - \hat{x}_{k|k-1}) h(xk​)≈h(x^k∣k−1​)+∂xk​∂h​∣∣∣∣∣​x^k∣k−1​​(xk​−x^k∣k−1​)

同样定义观测雅可比矩阵：

Hk≜∂h∂xk∣x^k∣k−1H_k \triangleq \frac{\partial h}{\partial x_k}\bigg|_{\hat{x}_{k|k-1}} Hk​≜∂xk​∂h​∣∣∣∣∣​x^k∣k−1​​

注意：EKF 里的 FkF_kFk​ 和 HkH_kHk​ 不再是固定的常数矩阵，而是在每一时刻的当前估计点处重新计算的雅可比矩阵。
这是 EKF 和 KF 最本质的区别。

# 3、EKF 的推导流程

有了局部线性化之后，EKF 的推导逻辑和 KF 完全一致，只是把所有的矩阵 FkF_kFk​、HkH_kHk​ 替换成了雅可比矩阵，把矩阵乘法替换成了函数调用。

# 预测步（Predict）

状态预测： 直接用非线性函数 fff 向前传播：

x^k∣k−1=f(x^k−1∣k−1,uk)\hat{x}_{k|k-1} = f(\hat{x}_{k-1|k-1}, u_k) x^k∣k−1​=f(x^k−1∣k−1​,uk​)

协方差预测： 用雅可比矩阵 FkF_kFk​ 来传播不确定性（和 KF 推导一模一样，但 FkF_kFk​ 现在是雅可比）：

Pk∣k−1=FkPk−1∣k−1FkT+QkP_{k|k-1} = F_k P_{k-1|k-1} F_k^T + Q_k Pk∣k−1​=Fk​Pk−1∣k−1​FkT​+Qk​

为什么协方差传播还能用这个公式？
因为我们已经在当前估计点处做了线性化，在这个局部邻域里，fff 的行为近似于线性映射 FkF_kFk​。线性映射对协方差矩阵的传播规则就是 FPFTF P F^TFPFT，这是概率论中的标准结论。
当然，这只是近似 —— 越非线性的系统，这个近似误差越大。

# 更新步（Update）

计算残差： 用观测函数 hhh 计算预测观测值，再与真实测量值作差：

y~k=zk−h(x^k∣k−1)\tilde{y}_k = z_k - h(\hat{x}_{k|k-1}) y~​k​=zk​−h(x^k∣k−1​)

计算误差协方差：

Sk=HkPk∣k−1HkT+RkS_k = H_k P_{k|k-1} H_k^T + R_k Sk​=Hk​Pk∣k−1​HkT​+Rk​

计算卡尔曼增益：

Kk=Pk∣k−1HkTSk−1K_k = P_{k|k-1} H_k^T S_k^{-1} Kk​=Pk∣k−1​HkT​Sk−1​

更新状态估计：

x^k∣k=x^k∣k−1+Kky~k\hat{x}_{k|k} = \hat{x}_{k|k-1} + K_k \tilde{y}_k x^k∣k​=x^k∣k−1​+Kk​y~​k​

更新误差协方差（Joseph Form）：

Pk∣k=(I−KkHk)Pk∣k−1(I−KkHk)T+KkRkKkTP_{k|k} = (I - K_k H_k) P_{k|k-1} (I - K_k H_k)^T + K_k R_k K_k^T Pk∣k​=(I−Kk​Hk​)Pk∣k−1​(I−Kk​Hk​)T+Kk​Rk​KkT​

卡尔曼增益的推导和 KF 完全一致（让 tr(Pk∣k)\text{tr}(P_{k|k})tr(Pk∣k​) 最小），这里不再重复。
这一步的全部公式形式和 KF 相同，唯一的区别是 HkH_kHk​ 是在 x^k∣k−1\hat{x}_{k|k-1}x^k∣k−1​ 处求值的雅可比矩阵。

# 4、完整迭代公式汇总

每个时刻重复以下五步：

Fk=∂f∂x∣x^k−1∣k−1,Hk=∂h∂x∣x^k∣k−1F_k = \frac{\partial f}{\partial x}\bigg|_{\hat{x}_{k-1|k-1}}, \quad H_k = \frac{\partial h}{\partial x}\bigg|_{\hat{x}_{k|k-1}} Fk​=∂x∂f​∣∣∣∣∣​x^k−1∣k−1​​,Hk​=∂x∂h​∣∣∣∣∣​x^k∣k−1​​

x^k∣k−1=f(x^k−1∣k−1,uk)\hat{x}_{k|k-1} = f(\hat{x}_{k-1|k-1}, u_k) x^k∣k−1​=f(x^k−1∣k−1​,uk​)

Pk∣k−1=FkPk−1∣k−1FkT+QkP_{k|k-1} = F_k P_{k-1|k-1} F_k^T + Q_k Pk∣k−1​=Fk​Pk−1∣k−1​FkT​+Qk​

Kk=Pk∣k−1HkT(HkPk∣k−1HkT+Rk)−1K_k = P_{k|k-1} H_k^T \left( H_k P_{k|k-1} H_k^T + R_k \right)^{-1} Kk​=Pk∣k−1​HkT​(Hk​Pk∣k−1​HkT​+Rk​)−1

x^k∣k=x^k∣k−1+Kk(zk−h(x^k∣k−1)),Pk∣k=(I−KkHk)Pk∣k−1\hat{x}_{k|k} = \hat{x}_{k|k-1} + K_k \left( z_k - h(\hat{x}_{k|k-1}) \right), \quad P_{k|k} = (I - K_k H_k) P_{k|k-1} x^k∣k​=x^k∣k−1​+Kk​(zk​−h(x^k∣k−1​)),Pk∣k​=(I−Kk​Hk​)Pk∣k−1​

对比 KF 的迭代公式，会发现结构完全一样，差别仅在：

  • Fkxk−1F_k x_{k-1}Fk​xk−1​ → f(xk−1,uk)f(x_{k-1}, u_k)f(xk−1​,uk​)（用函数替换矩阵乘法）
  • HkxkH_k x_kHk​xk​ → h(xk)h(x_k)h(xk​)（同上）
  • FkF_kFk​、HkH_kHk​ 每步重新计算雅可比，而非固定矩阵

# 5、EKF 的局限性

EKF 通过一阶线性化巧妙地把非线性问题 "化归" 为线性问题，但这个近似并非没有代价。

线性化误差
一阶泰勒展开忽略了高阶项。如果系统高度非线性（比如强烈的曲面旋转），这个误差会很大，导致滤波发散或估计偏差。

雅可比矩阵的计算成本
每一步都需要对 fff、hhh 求偏导并在当前估计点处求值。对于复杂系统，这可能需要解析推导并手动实现大量矩阵表达式，工程量不小，也容易出错。

旋转量的问题
三维空间中的旋转（SO (3) 群）不能直接用欧拉角或旋转矩阵放进 EKF，因为它们的参数化方式本身就会引入奇异性（万向锁）或过参数化。
这正是 ESKF（Error-State Kalman Filter，误差状态卡尔曼滤波） 诞生的核心动机 —— 把旋转量的微小扰动（误差状态）拿去做 KF，而把旋转的全局更新留给流形上的操作。

接下来，我们就来研究 ESKF。
